## 1. Paperclip

- Type: Repo
- Link: https://github.com/paperclipai/paperclip
- Homepage: https://paperclip.ing

### 正體中文
Paperclip 是一個開源的 AI 編排平台，目標是幫助使用者用多個 AI agents 來協作經營「幾乎零人工」的公司。它把任務管理、組織架構、預算控制、治理與 agent 協作整合在一起，適合想要長時間運行多個 agents、又希望持續追蹤工作與成本的人。

### English
Paperclip is an open-source AI orchestration platform for running near zero-human companies with coordinated AI agents. It combines task management, org structure, budgets, governance, and agent collaboration in one system, making it especially interesting for people who want to run multiple agents over time while keeping work and costs visible.

## 6. AI Night Shift

- Type: Repo
- Link: https://github.com/JudyaiLab/ai-night-shift
- Homepage: https://judyailab.com

### 正體中文
AI Night Shift 是一個多代理自主框架，目標是讓不同 AI agents 在離峰時間持續協作，把研究、開發、巡檢與整理工作留到「你在睡覺時」持續進行。它把多代理分工、共享協議、排程與 dashboard 結合起來，對想建立夜間自動化工作流的人很有參考價值。

### English
AI Night Shift is a multi-agent autonomous framework designed to let different AI agents keep working together during off-hours. It combines shared protocols, scheduling, role-based coordination, and a dashboard, making it a useful reference for anyone interested in overnight automation and long-running agent workflows.
